# EconoSuite Subagents Documentation Index

This directory contains the authoritative documentation and instructions for specialized subagents used within the EconoSuite orchestration pipeline. These subagents are activated for specific technical and domain tasks.

## 1. Core Experts
- [Econo-Expert](./econo-expert.md): Specialized economic modeling, data synthesis, and econometric reporting.
- [Hema-Expert](./hema-expert.md): Specialized hematology clinical researcher and medical writer.
- [Bkend-Expert](./bkend-expert.md): bkend.ai BaaS platform expert for auth and data modeling.

## 2. Technical Architects
- **Code-Analyzer**: Static analysis and architecture compliance.
- **CTO-Lead**: High-level technical direction and orchestration.
- **Frontend-Architect**: UI/UX design and design system management.
- **Security-Architect**: Vulnerability analysis and authentication design.
- **Infra-Architect**: AWS, Kubernetes, and Terraform management.

## 3. Product Management
- **PM-Lead**: PM team orchestration.
- **PM-PRD**: Synthesis of research into a comprehensive PRD.
- **PM-Discovery**: Opportunity Solution Tree (OST) analysis.
- **PM-Research**: Market sizing and competitor analysis.

## 4. Quality Assurance
- **QA-Strategist**: Test planning and verification coordination.
- **QA-Monitor**: Real-time Docker log monitoring (Zero Script QA).
- **PDCA-Iterator**: Automatic improvement loops for gap reduction.
